# CET-6 Data Library

这个目录用于沉淀 CET-6 学习资料，目标是把真题、答案、听力原文、写作与翻译训练材料整理成可长期复用的数据源。

## 目录结构

- `papers/` 真题正文或试卷文件
- `answers/` 参考答案与解析
- `transcripts/` 听力原文、听力材料说明
- `writing/` 写作题目、范文、句型模板、批改记录
- `translation/` 翻译题、参考译文、常见问题
- `sources/` 原始来源清单与可信度评估
- `index/` 总索引、年份映射、标签索引
- `daily-logs/` 每日真实训练记录
- `reviews/` 周复盘、阶段复盘、资料质量复盘

## 资料分级

- `A`：高可信，来源稳定，可直接用于训练
- `B`：可用，但需人工交叉验证
- `C`：仅作线索，不直接作为核心训练材料

## 命名规则

推荐文件名：

`<year>-<month>-set<1|2|3>-<type>.md`

示例：
- `2023-06-set1-paper.md`
- `2023-06-set1-answers.md`
- `2023-06-set1-transcript.md`

## 当前策略

1. 先建立目录和索引
2. 先沉淀来源，再逐步下载或整理正文
3. 优先补听力原文、答案、解析
4. 所有资料先入库，再进入训练流程
