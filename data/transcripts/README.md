# CET-6 Transcripts

这个目录用于存放**已实际整理或核验过**的听力原文与样本说明，不放纯链接清单。

## 使用规则

- 只有在完成“入口核验”后，才把样本信息写入本目录或其索引
- 若当前只有入口、还未确认具体年份/套卷/原文完整性，不要伪装成已入库正文
- 每个样本至少要写清：年份、批次、来源、用途、当前状态
- 若只是训练切片，也要说明来自哪一个入口，避免后续失去来源链路

## 推荐文件命名

- 完整原文：`<year>-<month>-set<1|2|3>-transcript.md`
- 样本说明：`<year>-<month>-sample-<topic>.md`

示例：
- `2024-12-set1-transcript.md`
- `2024-12-sample-news-listening.md`

## 当前策略

现阶段先不追求批量入库，而是先把“已核验可直接训练”的首批样本登记清楚，再逐步扩展正文。
