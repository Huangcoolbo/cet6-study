# CET-6 Reviews

这个目录用于存放真实训练产生的周复盘、阶段复盘和专项复盘。

使用规则：
- 周复盘：每 7 天 1 份，命名 `YYYY-Www-weekly-review.md`
- 阶段复盘：每 28 天 1 份，命名 `YYYY-MM-stage-review.md`
- 不提前伪造结论；没有数据就只建骨架，不写完成判断
- 复盘时必须引用：
  - `index/stage-progress-evaluation.md`
  - `index/weak-point-tracker.md`
  - `daily-logs/*.md`

当前用途：
- 先作为“实际运行记录”的落点
- 等首轮真实打卡数据积累后，再写第一份完整周复盘
