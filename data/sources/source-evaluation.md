# CET-6 Source Evaluation

## 评估标准

- 来源稳定性
- 内容完整性
- 是否含答案/解析
- 是否含听力原文
- 是否疑似转载/回忆版/用户上传
- 是否适合长期当数据源

## 当前已发现来源（第一轮）

### 1. TingClass
- URL: https://www.tingclass.net/show-5407-3662-1.html
- 类型: 听力真题/原文/答案页
- 价值: 对听力专项有直接价值
- 风险: 更像单页资料，不一定适合做完整年度总库
- 评级: B
- 结论: 可作为听力补充来源，适合先用于听力原文整理

### 2. 可可英语 KeKeEnglish
- URL: https://m.kekenet.com/cet6/ljtl/cet6zhenti/List_2.shtml
- URL: https://m.kekenet.com/cet6/ljtl/cet6zhenti/List_1.shtml
- 类型: 历年六级听力真题 / MP3 / 原文 / 讲解
- 价值: 对听力训练非常实用
- 风险: 主要覆盖听力，不是完整真题总库
- 评级: A-
- 结论: 优先作为听力资料库入口

### 3. 5xue
- URL: http://5xue.cc/?page_id=2735
- 类型: 历年六级听力真题入口页
- 价值: 可作为线索补充
- 风险: 站点权威性一般，需要交叉验证
- 评级: B-
- 结论: 仅作补充，不做核心来源

### 4. CSDN 汇总帖
- URL: https://blog.csdn.net/weixin_47095836/article/details/128310998
- 类型: 真题汇总转载
- 价值: 提供资料线索
- 风险: 二次转载、完整性和准确性不稳定
- 评级: C
- 结论: 不直接用于核心训练

### 5. 知乎专栏 / Scribd / Studocu / Bilibili
- 类型: 用户发布或转载内容
- 价值: 可作线索、补充讲解或字幕
- 风险: 非稳定、非标准化、版权和准确性风险更高
- 评级: C
- 结论: 不作为主数据源，只作辅助参考

## 当前结论

- 听力资料优先用：可可英语 + TingClass
- 真题完整卷与答案解析：继续筛更稳定来源
- 用户后续上传真题、答案、听力原文后，应优先纳入本地库
